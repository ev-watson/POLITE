# Windows Bridge Checklist

- [ ] ASCOM Platform installed
- [ ] Camera COM driver installed and validated
- [ ] ASCOM Remote Server installed and running
- [ ] Camera device added in Remote Server (device number noted)
- [ ] Alpaca port open in Windows Firewall (default 11111)
- [ ] `management/v1/description` reachable from client machine
- [ ] `camera_diag.py` succeeds from client machine
